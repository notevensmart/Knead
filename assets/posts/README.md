# Post Assets

Supplied Knead social post images are stored in this folder using these filenames:

- `i-miss-my-old-problems.png`
- `some-days-surviving-is-enough.png`
- `its-okay-to-take-up-space.png`
- `healed-people-heal-people.png`
- `progress-not-perfection.png`

These are client-supplied social media/post illustrations and should be treated as image assets. The homepage calm hero uses these files as decorative reflection cards.
