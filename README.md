# Knead Website

This repository now contains the selected Knead website direction based on Prototype 1.

## Local Hosting

From the repository root, run a simple static server:

```powershell
python -m http.server 4173
```

Then open:

```text
http://127.0.0.1:4173/
```

The production-ready root files are:
- index.html
- styles.css
- assets/
